# 학교생활기록부 점검기 (2026)

서술형 생기부(세특·창의적체험활동·행동특성)의 금지어·형식·사교육·고교블라인드·중복을
**브라우저 안에서만** 자동 점검하는 도구. 2026학년도 기재요령 기준.

## 완전 오프라인
개인정보가 많은 생기부 특성상 **인터넷 없이 동작**합니다.
`index.html`(또는 `생기부점검기.html`) 한 파일만 내려받아 더블클릭하면 됩니다.
xlsx·pdf.js 라이브러리와 PDF 워커까지 파일 안에 내장되어 있어, 어떤 데이터도 외부로 전송되지 않습니다.

- 배포본: <https://hong-jame.github.io/saengibu-checker/>

## 구조 (소스 ↔ 빌드)
- `src/index.html` — **편집용 소스**. 라이브러리 자리에 `<!--INLINE:*-->` 마커만 둔다.
- `vendor/` — 내장할 라이브러리(xlsx, pdf.js, pdf.worker).
- `build.mjs` — 마커에 vendor를 인라인해 **단일 파일** `index.html`·`생기부점검기.html` 생성.

## 빌드
```bash
node build.mjs
```
소스(`src/index.html`)를 고친 뒤 위 명령을 실행하면 배포용 단일 파일이 다시 만들어진다.
